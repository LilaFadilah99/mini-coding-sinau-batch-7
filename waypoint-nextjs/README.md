## рџљЂ Setup & Run

### 1. Install Dependencies

```bash
npm install
```

### 2. Setup Environment Variables

Copy `.env.example` jadi `.env.local`:

```bash
cp .env.example .env.local
```

Lalu edit `.env.local`:

```
GEMINI_API_KEY=AIza...
NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY=eyJ...
```

- **Gemini API Key**: dapatkan di https://aistudio.google.com/app/apikey
- **Supabase URL & Publishable Key**: Supabase dashboard в†’ Settings в†’ API

### 3. Setup Supabase Database

Di Supabase dashboard в†’ SQL Editor, run script ini:

## рџЋЁ Design System (Tailwind Tokens)

| `bg-bg-secondary` | `#f8fafc` | Section alternatif |
| `bg-bg-tertiary` | `#f1f5f9` | Highlight halus |

| `shadow-xl-soft` | floating cards | Hero card |
